# Situational Geometry Game Portal

A single-page HTML5 game for GitHub Pages.

## Files
- `index.html` — full game portal with canvas animation, countdown timer, sliders, labels, score, scenarios, and live metrics.

## Deploy to GitHub Pages
1. Create a GitHub repository.
2. Upload `index.html` to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose `main` and `/root`, then save.
6. Open the GitHub Pages URL after deployment finishes.

## Notes
This is an educational prototype. Scores are game metrics only.
